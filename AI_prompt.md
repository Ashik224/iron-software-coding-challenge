# AI Prompt
If I were to AI-prompt this task, I would have used **Cursor IDE**. The prompt is given below: <br>

- Consider that you are a Senior SE. Generate a method in C# that takes old keypad phone button presses as input and converts them to the corresponding alphabet as output. Applicable buttons are: digits from 2 to 9, '*' as backspace, '#' as send. Every input must end with a '#'. Digits 2 to 9 will have the exact alphabets that old phone keypads usually have. 
- Additionally, generate unit tests using xUnit to cover all the acceptance criteria and the edge cases for this method. 
- Create all the projects and classes with a proper folder structure.
- Provide comments where necessary.
- Create a readme.md file and provide the precise documentation. 
- While doing the whole thing, follow the standard coding convention.
